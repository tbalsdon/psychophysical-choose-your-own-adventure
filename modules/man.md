# Methods of manipulation

A core principle of experimental design is to measure the 'output' while varying one
'input' variable and keeping all else the same — so any change in the output can be
attributed to that input. But in psychological science there are complex processes
between input and output, occurring in a 'black box' we cannot directly access. To
make inferences about the mechanisms in that black box, we need to go beyond
measuring the input–output relationship: we **manipulate the mechanisms themselves**,
and measure how the input–output relationship *changes* after causing some change in
the processing.
